# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-12-25

### Added

- drawio plugin for Draw.io diagram creation and editing
- XML structure guide for .drawio files
- Shape and connector patterns (rectangle, ellipse, diamond, arrows)
- Japanese font (Noto Sans JP) configuration
- PNG/SVG/PDF export guide via drawio-export CLI
- Basic template file

## [0.1.0] - 2025-12-10

### Added

- Initial release of unity-development plugin
- unity-mcp-server tools usage guide
- VContainer DI patterns reference
- UniTask async processing guide
- Fail-Fast principle enforcement
- Troubleshooting guide
